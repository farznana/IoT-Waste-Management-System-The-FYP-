# IoT-Waste-Management-System-The-FYP-
Source code and system architecture for an IoT-based waste management system designed for high-rise apartment efficiency (FYP).

📌 **Project Overview**
This is my Final Year Project (FYP) for my Bachelor in ICT at MSU. The project addresses the challenges of waste management in high-density residential buildings by using IoT technology to monitor waste levels in real-time.

🚀 **Features**
- Real-time Monitoring: Uses sensors to track bin fullness.  
- Automated Alerts: Sends notifications when waste reaches a certain threshold.  
- Data Accuracy: Ensures reliable data collection for building management teams.

🛠 **Tech Stack**
- Hardware: Arduino/ESP32, Ultrasonic Sensors.  
- Software: C++, IoT Cloud Platforms.  
- Concepts: Data communication and automated reporting.
